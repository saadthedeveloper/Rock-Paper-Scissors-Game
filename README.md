# 🎮 Rock-Paper-Scissors Game

A fun and interactive Rock-Paper-Scissors game built with **HTML, CSS, and JavaScript**.  
Features a clean, modern design and dynamic score tracking for both the user and computer.

## ✨ Features
- 🎨 Beautiful, responsive HTML/CSS design
- 🖱️ Clickable choices with instant feedback
- 🔢 Live score updates for user and computer
- 🤖 Randomized computer choices
- 📜 Simple, readable JavaScript logic


## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📂 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/rock-paper-scissors.git

2. Play and enjoy!
